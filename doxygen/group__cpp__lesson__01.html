Lesson 01 – Tipos de datos, scope y calificadores

Objetivos

Al finalizar esta lección, el estudiante será capaz de:

  • Identificar los tipos de datos fundamentales en C++
  • Comprender el concepto de scope (alcance) de las variables
  • Diferenciar entre variables locales, globales y estáticas
  • Usar correctamente los calificadores static y const
  • Evitar errores comunes relacionados con vida útil y mutabilidad

Tipos de datos fundamentales

C++ es un lenguaje fuertemente tipado. Cada variable tiene un tipo que define memoria, operaciones válidas e interpretación del valor.

int contador = 0;
float voltaje = 3.3f;
bool activo = true;

Scope (alcance de las variables)

Scope local

void ejemplo() {
int x = 10;
}

Scope global

int g_contador = 0;
void incrementar() {
g_contador++;
}

Variables static

static dentro de funciones

void contador() {
static int veces = 0;
veces++;
}

static a nivel de archivo

static int archivoInterno = 42;

El calificador const

const int maxUsuarios = 10;

Combinación de const y static

static const int bufferSize = 256;
